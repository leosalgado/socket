## Install

Requirements:

- [**Go**](https://go.dev/) 1.24.1 or newer
- [**Git**](https://git-scm.com/downloads)

## Usage

Clone the repository: 

```sh
git clone https://github.com/leosalgado/socket.git
cd socket
```

Install dependencies:

```sh
go mod tidy
```

Run the ***server***:

```sh
go run main.go server
```

Run the ***client***:

```sh
go run main.go client
```
